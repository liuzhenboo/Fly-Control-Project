File list of the FDC 1.3.3 distribution package
===============================================
After installing the toolbox, all files from this list should be present in
your FDC 1.3 folders. If any of these files is missing, please contact the
author at the following contact address given at the bottom of this file.

Note: folder names in upper-case, file-names in lower-case. There is one
exception to this rule: the *.HLP files in the HELP folder are still in
upper-case (support for those files has been discontinued anyway, they have
only been retained for ex-FDC 1.2 users and users without an Internet
browser).
