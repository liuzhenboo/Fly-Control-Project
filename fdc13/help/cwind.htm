Subsystem Cwind: constant horizontal wind

Cwind is a masked SIMULINK system, which is used to calculate velocity components of a constant horizontal wind along the body-axes of the aircraft. Cwind can easily be enhanced for vertical wind- components as well if it is unmasked first. If you want the wind velocity to vary with altitude, use BLwind instead. Both Cwind and BLwind are contained in the blocklibrary WINDLIB.

The user must specify the wind velocity Vw in [m/s] and the wind direction psi_w in [rad].
Note: if the wind is blowing to the north, psi_w should be equal to pi rad!

Inputsignal (scalar)

  psi: heading of the aircraft [rad].

Outputvector

  ywind = [uw vw ww]'

  uw : wind velocity component along XB-axis [m/s]
  vw : wind velocity component along YB-axis [m/s]
  ww : wind velocity component along ZB-axis [m/s] (by default:
                                                          ww = 0)
