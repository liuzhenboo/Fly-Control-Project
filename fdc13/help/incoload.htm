The load utility INCOLOAD

INCOLOAD is a MATLAB routine which can be applied to load all kinds of model-data, trimmed flight conditions, linearized aircraft models, etc. from file. By default, trimmed flight conditions computed by ACTRIM are stored in files with extension *.TRI, linearized models obtained by ACLIN get the extension *.LIN, and model-parameter definition files (such as the one created by MODBUILD) get extension *.DAT, but the user is free to specify other file-types.
