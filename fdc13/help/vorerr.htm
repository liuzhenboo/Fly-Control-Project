The radio-navigation block VORerr

The masked subsystem block VORerr from the sublibrary VOR of the block-library NAVLIB can be used to implement steady-state VOR errors. The user must specify the required error percentage.

Note: the nominal VOR signals can be determined with the block VOR from the same block-library. The system VOR example demonstrates how these blocks can be combined.

Input signal

  Gamma_VOR (nominal value, coming from block VOR), [rad]
Gamma_VOR is the angle between the line through the projection point of the aircraft's c.g. on the ground and the VOR antenna, and the nominal VOR bearing (see also the references below).

Output signal

  Gamma_VOR (nominal value + steady-state error), [rad]
