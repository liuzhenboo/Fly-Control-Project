Load routine LOADER

LOADER is a MATLAB program which loads parameters for the non-linear aircraft model from file. By default it will search for *.DAT files. The aircraft model-data will normally be stored in AIRCRAFT.DAT.

These datafiles can be created with MODBUILD. This is only necessary if you want to implement data for a different aircraft, or change parameters of the current aircraft model.

Note: LOADER will also search for the vector xfix in the MATLAB workspace. If this vector can't be found it will be set to its default value. xfix is used by the gain-block XFIX within the aircraft model for artificially fixing state variables to their initial values.
