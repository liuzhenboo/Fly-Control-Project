The utility SYSTPROP

SYSTPROP computes the following properties of a linear system: 
     1. time constant                             tau
     2. natural frequency of undamped system      w0
     3. eigenfrequency of the system              wn
     4. period                                    P
     5. damping factor                            zeta
     6. percentage overshoot                      PO
     7. peak-time                                 Tpeak
     8. settling time                             Tset
     9. halve-time                                Thalve
Conclusions are displayed on screen and stored in the file SYSTPROP.DAT.

[X,Y] = systprop(A,B,C,D) or [X,Y] = systprop(A) will return the system properties in the matrix X and the eigenvalues of A in the columnvector Y. The matrix X is defined as: X = [tau, w0, wn, P, zeta, PO, Tpeak, Tset, Thalve].

[X,Y] = systprop(num,den) will evaluate the equivalent transfer function representation.

systprop(Y) returns the vector X if Y contains the eigenvalues of a system in a columnvector.
