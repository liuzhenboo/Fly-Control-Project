Non-linear aircraft model

The FDC toolbox contains a generalized non-linear aircraft model. The aircraft-dependent parts of this model were designed for the De Havilland 'Beaver' aircraft, but it shouldn't be too difficult to adapt the model for other types of aircraft. This model is stored in the SIMULINK system Beaver. Open-loop simulation models of the 'Beaver' aircraft

There are three different open-loop simulation models for the 'Beaver' aircraft (which can be applied to any aircraft model that uses the same inputs and outputs as the system Beaver). For each of these open-loop models there is also a 'tutorial' which explains the function of each block within these systems. These systems and tutorials have been listed in the following table:

Non-linear open-loop responses to control inputsOLOOP1
Non-linear open-loop responses to atmospheric turbulenceOLOOP2
Open-loop responses to control inputs, using a linearized aircraft modelOLOOP3
Tutorial for OLOOP1OLOOP1T
Tutorial for OLOOP2OLOOP2T
Tutorial for OLOOP3OLOOP3T


Another example of an open-loop application is the routine TRIMDEMO. There is no on-line help available for this routine yet; type help trimdemo at the MATLAB command-line for information about that routine.


Simulation models of the 'Beaver' autopilot

The FDC toolbox contains a complete non-linear model of the 'Beaver' autopilot, which includes options like gain-scheduling and longitudinal/lateral cross-coupling. Two simplified versions of the autopilot simulation model were included as well in order to speed-up computations:

Autopilot simulation without sensor models, radio-navigation signals, and noise signalsAPILOT1
Autopilot simulation models without noise signalsAPILOT2
Complete autopilot simulation modelAPILOT3


The systems PAH, RAH, and PAHRAH provide alternative implementations of the Pitch and Roll Attitude Hold modes. These systems haven't been documented here; see chapter 12 of the user-manual for more details.
