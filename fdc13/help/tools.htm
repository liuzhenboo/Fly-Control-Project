Analytical tools from the FDC toolbox

The FDC toolbox makes use of the standard SIMULINK integrators for simulations and two dedicated MATLAB programs for the determination of linearized aircraft models and steady-state trimmed flight conditions. The source files of these programs can be found in the FDC subdirectory TOOLS.
  • ACTRIM computes trimmed-flight conditions for the non-linear 'Beaver' model Beaver, or any other aircraft model that uses the same inputs and state variables as that system.

  • ACLIN linearizes the aircraft model around a user-specified operating point and gives the user the opportunity to select a subset of the state equations from the linearized model. It uses the SIMULINK program LINMOD for the actual linearization process.
The FDC subdirectory TOOLS also contains the routines LOADER and INCOLOAD, which facilitate initialization of the non-linear aircraft model, and RESULTS and RESPLOT, which simplify processing of simulation results. FIXSTATE is a companion program for the block XFIX within the system Beaver, NSWITCH creates graphical switch-blocks with n input signals, RECOVER solves problems due to a nasty SIMULINK bug, and SYSTPROP is a useful utility for extracting system-information from a linearized model.

Finally, a SIMULINK block-library FDCTOOLS has been included in the TOOLS subdirectory; it contains some useful general-purpose SIMULINK blocks. Type fdctools at the command-line to open it.
