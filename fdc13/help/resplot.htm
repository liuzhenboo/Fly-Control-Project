The utility RESPLOT

RESPLOT is a sample plot-macro that can be used after running RESULTS. It plots the most important time trajectories of states, inputs, and outputs for the non-linear aircraft model Beaver. The source-code of this program, which can be found in the file RESULTS.M in the FDC directory TOOLS, needs to be changed in order to plot other results or change the presentation of the simulation results. It is planned to improve this utility considerably for future versions of the toolbox (along with the utility RESULTS itself).
